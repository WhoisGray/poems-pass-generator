# مشارکت در Poems Pass Generator

از علاقه‌ات ممنونیم! 🌹 ساده‌ترین و باارزش‌ترین مشارکت، **افزودن شعر و ضرب‌المثل** به دیتابیس است.

## افزودن شعر یا ضرب‌المثل

فایل [`data/poems.json`](data/poems.json) را باز کن و به آرایه‌ی `poems` یا `proverbs` مدخل اضافه کن:

```json
{ "id": "nezami-1", "poet": "نظامی", "fa": "همه عالم تن است و ایران دل", "fg": "hame alam tan ast o iran del" }
```

- `id` — شناسه‌ی یکتا (مثلاً `poet-شماره`)
- `poet` — نام شاعر (برای ضرب‌المثل لازم نیست)
- `fa` — متن فارسی (فقط برای نمایش)
- `fg` — **آوانگاری فینگلیش**؛ مبنای ساخت گذرواژه است. فقط حروف لاتین کوچک و فاصله.

### تولید خودکار `fg`
اگر آوانگاری را بلد نیستی، می‌توانی با ماژول داخلی حدس بزنی:

```bash
node -e "import('./src/core/transliterate.js').then(m => console.log(m.transliterate('متن فارسی شما')))"
```

سپس خروجی را بازبینی و در صورت نیاز اصلاح کن (آوانگاری خودکار حدسی است).

## قبل از ارسال Pull Request

```bash
npm test          # باید همه سبز باشد
node bin/poems-pass.js --stats   # دیدن آمار جدید دیتابیس
```

## قواعد
- بدون افزودن هیچ وابستگی خارجی (پروژه عمداً zero-dependency است).
- یک مدخل = یک بیت/مصرع کوتاه (نه کل غزل).
- از اشعار با مالکیت عمومی (کلاسیک) استفاده کن؛ برای شاعران معاصر به حق نشر دقت کن.
